  • {Coordinate} clone()
    Clones this instance.

    {Coordinate} A point instance cloned from this.
  • {number} compareTo(other)
    Compares this Coordinate with the specified Coordinate for order. This method ignores the z value when making the comparison. Returns:
    • -1 : this.x < other.x || ((this.x == other.x) && (this.y < other.y))
    • 0 : this.x == other.x && this.y = other.y
    • 1 : this.x > other.x || ((this.x == other.x) && (this.y > other.y))
    Note: This method assumes that ordinate values are valid numbers. NaN values are not handled correctly.
    Parameters:
    {Coordinate} other
    the Coordinate with which this Coordinate is being compared.
    Returns:
    {number} -1, zero, or 1 as explained above.
  • {number} distance(p)
    Computes the 2-dimensional Euclidean distance to another location. The Z-ordinate is ignored.
    Parameters:
    {Coordinate} p
    a point.
    Returns:
    {number} the 2-dimensional Euclidean distance between the locations.
  • {boolean} equals(other)
    Returns true if other has the same values for the x and y ordinates. Since Coordinates are 2.5D, this routine ignores the z value when making the comparison.
    Parameters:
    {Coordinate} other
    a Coordinate with which to do the comparison.
    Returns:
    {boolean} true if other is a Coordinate with the same values for the x and y ordinates.
